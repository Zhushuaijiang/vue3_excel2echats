# 功能详细说明

## 1. 数据看板 (Dashboard)

### K线图
- 展示货运量的开盘、收盘、最高、最低价格
- 支持时间范围选择
- 交互式缩放和平移

### 趋势图
- 多维度趋势展示
- 平滑曲线渲染
- 点击查看历史数据详情弹窗

### 历史数据查看
- 点击图表数据点弹出详情
- 表格展示历史记录
- 支持数据筛选和排序

## 2. 表单配置 (Form Config)

### 字段类型支持
- **文本类型**: 单行文本输入，支持字符串验证
- **数字类型**: 数字输入，支持小数、整数、范围校验
- **日期类型**: 日期选择器，支持日期范围
- **选择类型**: 下拉选择框，支持单选和多选

### 配置管理
- 创建自定义表单配置
- 编辑现有配置
- 删除不需要的配置
- 表单字段动态增删

### 使用场景
- 航空货运单录入
- 货物信息登记
- 运单状态跟踪
- 客户信息管理

## 3. 数据录入 (Data Entry)

### 动态表单
- 基于配置自动生成表单
- 字段类型自动验证
- 实时错误提示

### 数据管理
- 列表展示所有记录
- 快速搜索和筛选
- 批量操作支持
- 数据导出功能（待开发）

## 4. 大屏展示 (Big Screen)

### 实时刷新
- 可配置刷新间隔：30秒、1分钟、5分钟
- 类似心电图滚动刷新
- 平滑的动画效果

### 拖拽配置
- 拖拽添加图表组件
- 调整组件位置和大小
- 实时预览布局
- 保存配置

### 图表类型
- **折线图**: 适合趋势展示
- **柱状图**: 适合数据对比
- **饼图**: 适合占比分析

### 全屏模式
- 一键全屏展示
- 隐藏工具栏
- 专注数据可视化

## 5. AI 模板 (AI Templates)

### 提示语模板管理
- 创建自定义提示语模板
- 使用 `{user_input}` 占位符
- 模板分类和标签

### AI 生成示例

**模板：货运量分析**
```
请生成一个展示航空货运量的统计大屏，包含：
1. 总体货运量趋势图（折线图）
2. 各航线货运量对比（柱状图）
3. 货运量占比分析（饼图）
根据用户输入：{user_input}
```

**使用方法**：
1. 选择AI模板
2. 输入具体的分析需求
3. 点击生成按钮
4. AI自动生成大屏配置

## 6. 数据库

### SQLite 优势
- 本地存储，无需服务器
- 轻量级，占用资源少
- 支持百万级数据
- 高效的查询性能

### 表结构

#### form_configs
存储表单字段配置

#### cargo_records
存储货运记录数据

#### dashboard_configs
存储大屏配置

#### ai_templates
存储AI提示语模板

## 使用建议

### 初次使用
1. 先在"表单配置"创建表单字段
2. 到"数据录入"添加一些测试数据
3. 在"数据看板"查看数据可视化效果
4. 配置"大屏展示"用于演示

### 日常使用
- 定期录入新数据
- 查看趋势图表分析
- 使用大屏进行数据展示
- 根据需求调整配置

### 高级功能
- 创建多个表单配置
- 组合多个图表分析
- 使用AI自动生成报告
- 自定义刷新频率
