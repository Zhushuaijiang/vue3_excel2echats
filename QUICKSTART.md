# 快速开始指南

## 5 分钟快速上手

### 第一步：启动应用

```bash
npm install
npm run dev
```

应用窗口自动打开，您会看到主界面。

### 第二步：创建表单配置

1. 点击左侧菜单 **"表单配置"**
2. 点击 **"新增配置"** 按钮
3. 输入配置名称，例如：`货运单信息`
4. 点击 **"添加字段"** 添加以下字段：

| 字段名 | 类型 |
|--------|------|
| 运单号 | 文本 |
| 货物重量 | 数字 |
| 起飞日期 | 日期 |
| 航线 | 选择 |
| 目的地 | 文本 |

5. 点击 **"保存"** 完成配置

### 第三步：录入数据

1. 点击左侧菜单 **"数据录入"**
2. 在下拉框中选择刚才创建的 `货运单信息`
3. 填写表单数据
4. 点击 **"提交"** 保存

### 第四步：查看看板

1. 点击左侧菜单 **"数据看板"**（首页）
2. 查看K线图和趋势图展示
3. 点击图表上的数据点查看历史详情

### 第五步：配置大屏

1. 点击左侧菜单 **"大屏展示"**
2. 点击 **"配置大屏"** 按钮
3. 从左侧拖拽组件到中间区域：
   - 拖拽"折线图" → 放置到合适位置
   - 拖拽"柱状图" → 添加到页面
   - 拖拽"饼图" → 完成布局
4. 设置刷新间隔为 **"30秒"**
5. 点击 **"全屏显示"** 按钮

### 第六步：使用AI功能

1. 点击左侧菜单 **"AI模板"**
2. 查看系统预置的AI模板：
   - 货运量分析
   - 航线效率分析
   - 成本收益分析
3. 在大屏页面点击 **"AI生成"** 按钮
4. 选择模板并输入需求
5. 点击生成（当前为演示模式）

## 常用操作

### 查看已有配置

- **表单配置**: 查看所有自定义表单
- **数据录入**: 查看所有录入记录
- **AI模板**: 查看和管理AI提示语

### 编辑配置

点击表格中的 **"编辑"** 按钮可以修改：
- 表单字段
- AI模板内容

### 删除数据

点击 **"删除"** 按钮，确认后移除：
- 表单配置
- 数据记录
- AI模板

⚠️ **注意**: 删除表单配置会关联删除该配置下的所有数据记录。

## 常见问题

### 应用启动很慢？

这是正常现象，首次启动需要编译。后续启动会更快。

### 数据保存在哪里？

数据保存在本地SQLite数据库：
- Windows: `%APPDATA%/air-cargo-analytics/cargo_data.db`

### 如何备份数据？

直接复制数据库文件即可完成备份。

### 支持导入数据吗？

当前版本暂不支持批量导入，后续版本会添加Excel/CSV导入功能。
