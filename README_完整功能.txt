🚀 快速开始
==================================================

第一步：安装依赖
   npm install

第二步：初始化数据
   npm run init:all
   （自动创建8个表单模版 + 200-400条模拟数据）

第三步：配置AI（可选）
   copy electron\config.json.example electron\config.json
   编辑填入DeepSeek API Key

第四步：启动应用
   npm run dev

==================================================
📊 数据功能
==================================================

数据生成：
  - 8个表单模版：国际/国内货运单、仓储、航班等
  - 模拟数据：覆盖最近90天
  - 城市分布：10个主要城市
  - 货物类型：电子产品、服装、食品等
  - 数据量：200-400条记录

图表展示：
  - K线图：6个月货运波动
  - 趋势图：30天趋势变化
  - 折线图：15天详细数据
  - 柱状图：城市货运排名
  - 饼图：货物类型分布

==================================================
🎯 核心操作
==================================================

数据看板：
  1. 查看K线图和趋势图
  2. 点击图表查看历史详情
  3. 点击刷新更新数据

大屏展示：
  1. 点击"配置大屏"添加图表
  2. 按住头部拖拽移动位置
  3. 点击⛶按钮调整大小
  4. 点击"保存布局"保存
  5. 设置刷新间隔
  6. 全屏展示

AI生成：
  1. 点击"AI生成"按钮
  2. 选择模板
  3. 输入需求
  4. 自动生成布局
