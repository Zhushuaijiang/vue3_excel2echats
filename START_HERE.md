# 🚀 从这里开始！

## 欢迎使用航空货运统计和分析软件

恭喜！您已经获得了一个功能完整的航空货运数据分析软件。

## ⚡ 3步快速启动

### 步骤 1️⃣：安装依赖

打开终端，在项目目录下运行：

```bash
npm install
```

⏱️ **预计时间**: 3-5分钟

💡 **提示**: 如果 better-sqlite3 安装失败，请查看 `INSTALL.md` 的常见问题部分

### 步骤 2️⃣：启动开发

```bash
npm run dev
```

⏱️ **预计时间**: 10-30秒

🎉 **成功后**: Electron 窗口将自动打开，显示应用界面

### 步骤 3️⃣：开始使用

按照左侧菜单依次体验：

1. 📊 **数据看板** - 查看图表展示
2. ⚙️ **表单配置** - 创建表单配置
3. 📝 **数据录入** - 录入测试数据
4. 📺 **大屏展示** - 配置大屏布局
5. 🤖 **AI模板** - 管理AI提示语

## ⚠️ 重要提示

### 关于Linter错误
如果在安装依赖前打开项目，会看到TypeScript错误：

```
Cannot find module 'vue'
Cannot find module 'element-plus'
...
```

**这是正常的！** 运行 `npm install` 后这些错误会自动消失。

### 关于AI功能
当前AI功能处于**框架完成**状态，实际AI服务尚未集成。在大屏页面使用AI生成功能会显示提示消息。

### 关于数据库
- 数据自动保存在：`%APPDATA%/air-cargo-analytics/`
- 首次运行会自动创建数据库
- 已内置3个AI模板示例
